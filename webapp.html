Web app (iOS simulation)

This interactive mobile simulation replicates an iPhone environment, built using HTML, CSS and JavaScript. The project focuses on interface structure, user interaction logic and visual realism and is fully interactive.

This project showcases my ability to create web applications with the help of Arteficial Intelligence. Most of the code was generated by different AI chat models.

By building this project I learned how to create and optimise promts in order to get the desired result from the AI chat models. Additionally, by editing the code, I learned the basics of HTML, CSS and JavaScript.

The simulation can be started by swiping up from the white line at the bottom of the screen (as in a real iPhone). Users are led to a password screen. The password is "123456". After entering a correct password, users are led to the main screen of the simulation.

All apps except for voice memos are active. They are not fully developed, but they are interactive and can be used to test the user interface and user experience.

An additional improvement that I plan to make is to make all apps fully responsive and mimic most of the features of the real apps. Moreover, I plan to add more apps to the simulation.
