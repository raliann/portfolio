Graphics

A collection of visual projects spanning animation, illustration, and print design.

Chemistry Animation

An animation, visualising the ion exchange reaction between iron and sulfuric acid.

This animation was created frame by frame in Canva. It taught me the basics of animation and how to create smooth transitions between frames.

Thanks to this project I improved my dynamic graphic creating skills and learned the basics of animated videos and how they are created.

Biology Flipbook

An interactive digital flipbook containing 2 lessons for new knowledge on the topic of genetics, and one practice lesson, containing interactive exercises that help the students to consolidate their knowledge. The flipbook is designed for an engaging and visual reading experience.

The first stage was to create a pdf version of the students book. I created the design, layout and all images from scratch in Canva.

The second stage was to create an interactive flipbook from the pdf version. I used Heyzine to create an interactive flipbook from the pdf version.

The third stage was to create interactive exercises for the practice lesson. I used a dedicated educational software, called learningapps.org, to create interactive exercises for the practice lesson. The exercises can be opened directly in the flipbook, or in a new window in order to be full-screen.

Biology Brochure

A print-ready biology brochure created to the promote my interactive flipbook.

The brochure is made as a documentation, explaining the technical aspect of how the students book should be used. It includes the main purpose of the project, content of the flipbook and advantages of using it. Additionally, there is a qr code, that can be scanned to access the flipbook directly.

All of the graphics in the brochure were created from scratch in Inkscape and Canva. The brochure was then created in Microsoft Publisher and exported as a pdf.

Mountain Illustration

A vector landscape illustration created with precision and a clean aesthetic, showcasing skills in digital illustration.

By creating this image, I developed a strong understanding of vector-based design principles. I learned how Bézier curves, nodes, paths, and layers work together to form scalable graphics, and how shapes can be constructed and refined with precision. Working with gradients, opacity, and path operations (union, difference, intersection) helped me understand how complex visuals are built from simple geometric elements.

This process also strengthened my sense of composition, color harmony, and visual depth. Beyond design skills, I improved my patience and attention to detail, as vector illustration requires iterative refinement and deliberate control over every anchor point.
